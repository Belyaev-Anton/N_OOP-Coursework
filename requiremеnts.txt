Обязательные данные которые нужно внести для корректной работы программы:
	vk_user_id = {id пользователя}
	vk_token = {Токен VK}
	yandex_token = {Токен Яндекс диска}
	count_foto = (необходимое колличество фотографий которое надо скачать)

После запуска программы начинается процесс:
	поиск картинок (аватарок) пользоватля
	определение URL максимального размера картинки
	определение имени файла картинки
		если кол-во лайков файла не совпадает с кол-во лайков остальных файлов то - id фото + Кол-во лайков.
		если кол-во лайков файла совпадает с кол-во лайков остальных файлов то - id фото + Кол-во лайков + дата загрузки.

Потом проверка - существует ли папка с заданым именем - {yandex_folder_name} на яндекс диске? 
	Если ДА то удаляем ее

На Яндекс Диске в коневой зоне создается папка - {yandex_folder_name} по умолчанию - "0_Belyaev_A_A_VK_foto_profile"

Запускается процесс загрузки фото на Яндекс Диск в созданную папку {yandex_folder_name}

Создается файл {file_name_json}  в корневой папке на ПК где расположен файл с кодом программы

Файл {file_name_json} сохраняется на Яндекс диске в заданной папке {yandex_folder_name}

Удаляется файл {file_name_json} с ПК
